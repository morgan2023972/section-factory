# Release Notes - v1.5.0

Date: 2026-03-30

## Nouveautes

- Introduction du module sectionOptimizer (MVP assistant) pour optimiser des sections existantes.
- Nouvelle commande CLI:
  - `npm run optimize -- <file-path> [options]`
- Mode par defaut: rapport uniquement (aucune ecriture disque sans `--write`).

## sectionOptimizer (MVP)

Nouveaux modules:

- `src/core/sectionOptimizer/index.ts`
- `src/core/sectionOptimizer/cleanup.ts`
- `src/core/sectionOptimizer/minifier.ts`
- `src/core/sectionOptimizer/patterns.ts`
- `src/core/sectionOptimizer/safetyAudit.ts`

Capacites MVP:

- Nettoyage de code (proprete)
- Minification conservative CSS/JS (legerete)
- Suggestions de reutilisabilite (patterns)
- Audit cross-theme (securite)
- Rollback automatique si regression de validation

## CLI optimize

Options principales:

- `--cleanup`
- `--patterns`
- `--minify`
- `--safety`
- `--size-threshold <num>`
- `--write`
- `--output <path>`
- `--format text|json`

Succes d optimisation (par axe):

- Taille: succes si gain >= seuil de compression/cleanup
- Securite: succes si pattern risque supprime
- Structure: succes si regle de conformite appliquee

Codes de sortie:

- `0`: succes
- `1`: issue safety de severite high detectee
- `2`: erreur usage/IO
