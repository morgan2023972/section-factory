# Release Notes - v1.2.0

Date: 2026-03-30

## Nouveautes

- Simplification des commandes de discovery CLI autour de `--list-sections`.
- Suppression de l alias `--list-section` (singulier).
- Conservation de `--list-types` en mode compatibilite legacy avec warning de depreciation.

## Deprecations

- `--list-types` est deprecie.
- La suppression de `--list-types` est prevue dans 2 releases.
- Commande recommandee: `--list-sections`.

## Qualite

- Mise a jour des tests unitaires CLI pour verifier le warning de depreciation.
- Validation de la suite: tests unitaires passes.

## Documentation

- README, ARCHITECTURE et documentation projet alignes sur la nouvelle politique discovery.
